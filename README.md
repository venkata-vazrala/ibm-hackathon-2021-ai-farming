# AI-Assisted Farming — Crop Recommendation & Yield Prediction

**IBM Hack Challenge 2021** (SBSPS Challenge #5361) — shortlisted at Level 2. A web application that recommends the best crop to cultivate based on climate and soil-nutrient parameters, and predicts the expected yield and revenue.

## Tech Stack

IBM Watson Studio, AutoAI, machine learning, web-based GUI.

## Problem

Farmers largely rely on traditional, experience-based decisions about what to plant, with limited digital tooling to account for climate variability, soil nutrients, and market outcomes — increasing risk in an already high-risk industry.

## Solution

An application that:
1. Recommends the best crop(s) to cultivate for a given plot, based on climatic parameters and soil nutrient features, using an IBM Watson Studio / AutoAI-built ML model.
2. Predicts the expected yield and revenue for the cultivated land, surfaced through an interactive web interface.
